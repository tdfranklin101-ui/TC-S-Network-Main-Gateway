VIDEO SERVER SETUP - 3 Steps Per Video
==========================================

You'll create 3 new Replit projects, one for each video.

FOR EACH VIDEO SERVER:

1. CREATE NEW REPLIT PROJECT
   - Go to replit.com
   - Click "Create Repl"
   - Choose "Node.js" template
   - Name it: plant-the-seed-video (or we-said-so-video, or podcast-discussion-video)

2. COPY FILES TO NEW PROJECT
   - Copy all files from the corresponding folder:
     • video-servers/plant-the-seed-video/* → plant-the-seed-video Repl
     • video-servers/we-said-so-video/* → we-said-so-video Repl
     • video-servers/podcast-discussion-video/* → podcast-discussion-video Repl
   
3. COPY VIDEO FILE
   - From THIS project's public/videos/ folder:
     • Copy plant-the-seed.mp4 → plant-the-seed-video Repl
     • Copy we-said-so-monazite.mp4 → we-said-so-video Repl
     • Copy podcast-discussion.mp4 → podcast-discussion-video Repl

4. RUN & DEPLOY
   - Click "Run" in each Repl
   - Test the video plays
   - Click "Deploy" button
   - Copy the deployment URL (e.g., https://plant-the-seed-video.username.repl.co)

5. UPDATE MAIN PLATFORM
   - Come back to THIS project
   - Update public/main-platform.html with your 3 deployed URLs
   - Look for <!-- PLACEHOLDER URLS --> comment

VIDEO FILE SIZES (all under 500MB limit):
- plant-the-seed.mp4: 176MB ✅
- we-said-so-monazite.mp4: 231MB ✅
- podcast-discussion.mp4: 134MB ✅

After setup, your main platform will link to these external video servers,
just like how it used to link to YouTube!
