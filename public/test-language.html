Language Translation Test Page

This page is used to verify that the language translation component is working correctly across the website.

Sample Text in Multiple Languages

The following text should be translated when you select a different language:

  • Welcome to The Current-See website.
  • We are building a solar-backed global economic system.
  • Join us in creating a more equitable financial future.
  • Every day, solar energy creates new value.
  • This value can be distributed to everyone on Earth.

Testing Rich Content

Complex content with formatting should also translate correctly:

Our Mission

The Current-See aims to revolutionize global economics by creating a solar-backed currency system that distributes daily value to all participants.

How It Works

  1. Solar panels generate electricity across the globe
  2. This electricity has a monetary value
  3. We track this value in real-time
  4. Every participant receives a daily distribution

Technical Implementation

Our language translation uses Google Translate API and includes the following features:

  • Dropdown menu with country flags
  • Support for multiple languages
  • Persistent language selection via localStorage
  • RTL language support
  • Mobile-friendly design
// Usage of the language translator document.addEventListener('DOMContentLoaded', () => { window.languageTranslator.init(); }); // Listen for language changes document.addEventListener('languageChanged', (e) => { console.log(`Language changed to: ${e.detail.language}`); });
