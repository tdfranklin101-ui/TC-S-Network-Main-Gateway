Open Standard v1.0

Solar Protocol v1.0 — Open Energy Equivalence Standard

1 Solar = 4,913 kWh. The Solar Protocol defines a universal reference for energy-based valuation across AI, compute, and digital assets. It embeds human-rights alignment and ecological accountability directly into machine-readable metadata.

Purpose

To create a transparent, auditable, and ethical baseline for energy accounting in the age of artificial intelligence and distributed compute. Each Solar represents a day's worth of clean, renewable human energy income.

JSON Specification

Download the SolarStandard.json file for direct API integration.

Usage Example

{
  "energy_consumed_kWh": 9826,
  "solar_equivalent": 2,
  "verification": "REC",
  "renewable_source": "SOLAR"
}

API Integration

Endpoint: GET /api/solar?kWh={value}

Example: /api/solar?kWh=9826

Response:

{
  "kWh": 9826,
  "solar_equivalent": 2,
  "unit": "Solar",
  "reference": "Solar Standard v1.0"
}

Key Fields

  • asset_id: Unique identifier (UUID or hash)
  • asset_type: AI_MODEL | DATA_CENTER | DIGITAL_ARTIFACT | TOKEN
  • energy_consumed_kWh: Numeric value of energy consumed
  • solar_equivalent: energy_consumed_kWh / 4913
  • renewable_source: SOLAR | WIND | HYDRO | MIXED | UNKNOWN
  • verification: PPA | REC | TC-S_LEDGER | SELF_REPORTED
