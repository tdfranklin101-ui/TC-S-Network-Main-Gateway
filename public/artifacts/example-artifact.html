How It Works

  1. Artifact creator runs the Solar index generator with energy footprint data
  2. Generator creates JSON-LD structured data at /solar-index/{id}.json
  3. Generator updates the Atom feed at /SolarFeed.xml
  4. This HTML page embeds the JSON-LD for AI/search engine discovery
  5. Energy footprint is permanently indexed and verifiable

Generator Command Used

node SolarStandard/generators/solar-index.js \
  --id=example-ai-model-001 \
  --type=AI_MODEL \
  --kwh=4913 \
  --name="Example GPT-4 Training Run" \
  --verification=REC \
  --source=SOLAR \
  --geo=US-CA
